# Related MSYS2 Configuration

1.断网安装msys2
2.更新包管理器

```
pacman -Syu
```

3.安装cmake/xmake

```
pacman -S cmake
pacman -Sy mingw-w64-x86_64-xmake
```

还有make

```
pacman -S make
```

4.安装arm-none-eabi-gcc

```
pacman -S mingw-w64-x86_64-arm-none-eabi-gcc
```
